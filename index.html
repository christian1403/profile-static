Tentang Saya

Saya adalah seorang mahasiswa aktif di Institut Teknologi Adhi Tama Surabaya (ITATS) yang saat ini berada di semester 4. Berdomisili di Surabaya, saya memiliki minat dan pengalaman yang kuat di bidang web development. Saya terbiasa mengembangkan aplikasi berbasis web menggunakan berbagai teknologi seperti HTML, CSS, JavaScript, PHP (CodeIgniter & Laravel), serta Node.js dengan Express.js. Selain itu, saya juga memiliki pengalaman dalam pengelolaan database menggunakan PostgreSQL dan MySQL. Dengan semangat belajar yang tinggi, saya terus mengembangkan kemampuan untuk menciptakan solusi digital yang efektif dan efisien.
